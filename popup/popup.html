Setup Guide

Step 1: Install Plugin in Obsidian

  1. Open Obsidian → Settings → Community plugins → Browse
  2. Search "Local REST API" → Install → Enable
  3. Click the gear icon ⚙️ next to it
  4. Copy the API Key
  5. Enable "Non-encrypted (HTTP) server" if not enabled

Step 2: Connect Brain Extension

  1. Click Brain icon in Chrome
  2. Paste your API Key
  3. Port: 27123 (HTTP) or 27124 (HTTPS)
  4. Click "Connect" then "Test" to verify

Step 3: Capture Chats

  1. Go to ChatGPT, Claude, Gemini, or Grok
  2. Click "Capture to Brain" button
  3. Check your Obsidian vault → AI-Brain/ folder

Connection Flow

Chrome Extension → HTTP → localhost → Obsidian Plugin → Your Vault

Troubleshooting

  • Connection failed? Make sure Obsidian is open and plugin is enabled
  • HTTP not working? Enable "Non-encrypted (HTTP) server" in plugin settings
  • HTTPS not working? Check the "Use HTTPS" checkbox and use port 27124
  • Files not appearing? Check your vault's AI-Brain/ folder
