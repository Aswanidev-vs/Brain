Setup

Two installs, five minutes

One plugin for Obsidian. One extension for Chrome. That's it.

Obsidian Plugin

1

Open Community Plugins

In Obsidian, go to Settings → Community plugins → Browse

2

Install Local REST API with MCP

Search for "Local REST API with MCP" by Adam Coddington. Install it, then enable it.

3

Copy your API Key

Click the gear icon next to the plugin. Copy the API key from the top. dont copy the word Bearer , copy the key.

4

Enable HTTP Server

Scroll down, enable "Non-encrypted (HTTP) server". Port should be 27123.

Chrome Extension

1

Load the Extension

Go to chrome://extensions/, enable Developer mode, click Load unpacked, select the Brain folder.and also pin this extension better access.

2

Paste your API Key

Click the Brain icon, paste the key you copied from Obsidian.

3

Connect

Port: 27123. Click Connect, then Test. You should see "Connected!".

4

Start Capturing

Open any supported AI chat. Click "Capture to Brain" or let auto-capture do its thing.

Troubleshooting

Common issues and how to fix them.

"Connection failed" when clicking Connect
Make sure Obsidian is open and the "Local REST API with MCP" plugin is enabled. Check that the API key matches what's in the plugin settings.
"API error 401: Authorization required"
The API key is wrong. Go back to Obsidian → "Local REST API with MCP" settings → copy the full key again. No extra spaces.
"Capture failed" or button doesn't work
Refresh the AI chat page. The extension needs to reload after installation or updates.
Auto-capture doesn't save anything
Open Brain settings, make sure "Enable auto-capture" is checked, and click Save Settings. Also check that your delay is not too high and wait for the AI reply to appear before expecting a save.
Claude auto-capture misses a reply sometimes
Claude updates its page differently from the other platforms, so Brain uses adaptive detection there. If Claude changes its UI and a reply is missed, wait for the response to settle or use the manual Capture to Brain button.
Notes don't appear in Obsidian
Check your vault's AI-Brain folder. If it doesn't exist, the connection might not be working — try the Test button again.
What do the colored dots on the Capture button mean?
The dot shows your live connection to Obsidian: Green = Connected, Red = Disconnected, Gray = Checking. The extension icon badge also shows ✓ or ✗.
"Extension context invalidated" error
This happens after the extension updates. Just refresh the page and try again.
